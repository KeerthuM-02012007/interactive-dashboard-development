# Interactive Dashboard Development

A modern, interactive dashboard application for data visualization and analytics.

## Project Structure

```
interactive-dashboard-development/
│
├── index.html           # Main HTML file
├── style.css            # Styling and layout
├── script.js            # JavaScript functionality
│
├── data/
│   └── dashboard_data.csv   # Sample dataset
│
├── assets/
│   ├── logo.png         # Project logo
│   └── screenshots/     # Screenshot directory
│
└── README.md            # Project documentation
```

## Features

- Interactive data visualization
- Responsive design
- Real-time data updates
- CSV data import support

## Getting Started

1. Clone this repository
2. Open `index.html` in your web browser
3. The dashboard will automatically load and display data from `dashboard_data.csv`

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- CSV data format

## File Descriptions

- **index.html**: Main entry point with dashboard layout
- **style.css**: Responsive styling and grid layout
- **script.js**: Dashboard logic and data handling
- **data/dashboard_data.csv**: Sample data for visualization
